# Authentication Kata

Build an authentication service with sign in, sign up, reset password functionality and with service a html frontend

## Requirements

* When the user opens the page the login is shown
* There is a username input field, which is limited to 20 characters.
* There is a password field, which is limited to 20 characters.
* There is a "Log in" button
* If login wasn't successfully an error is shown
* User has a link to sign up page and create a new user with username, email, password
* After sign up a welcome mail is sent to the user
* User has the option to reset the password with username and email
* If user password has been reset the temporary password is sent to the user
